# Portfolio Simulation

A simple Monte Carlo portfolio simulator built with Streamlit.

You can adjust:

* market returns and volatility
* a higher-risk allocation
* leverage
* time horizon

The app then simulates different portfolio paths and shows possible outcomes.

## Live App

https://portfolio-simulation-brian-young.streamlit.app/

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
